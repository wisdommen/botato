    **Step 1 (Python port): Create `tests/sim/adaptive_controller.py`**

    The AdaptiveController class mirrors the GDScript AdaptiveWeightController exactly:

    ```python
    class AdaptiveController:
        DEFENSIVE = ["projectile", "boss", "bumper"]
        OFFENSIVE = ["consumable", "item", "tree"]
        EMA_RATE = 0.02
        MULT_MIN = 0.7
        MULT_MAX = 1.3

        def __init__(self):
            self._reset()

        def _reset(self):
            self.ema_damage_rate = 0.0
            self.ema_health_ratio = 1.0
            self._prev_health = -1.0
            self._prev_wave_time_left = 1.0
            self._multipliers = {t: 1.0 for t in self.DEFENSIVE + self.OFFENSIVE}

        def get_multiplier(self, force_type: str) -> float:
            return self._multipliers.get(force_type, 1.0)

        def update(self, cur_hp: float, max_hp: float, wave_time_left: float, delta: float = 1.0/60.0):
            # Wave boundary check per D-03: prev < 0.05 AND current > 1.0
            if self._prev_wave_time_left < 0.05 and wave_time_left > 1.0:
                self._reset()
                self._prev_wave_time_left = wave_time_left
                return
            self._prev_wave_time_left = wave_time_left
            self._update_ema(cur_hp, max_hp, delta)
            self._update_multipliers()
    ```

    For `_update_ema`: Per D-01, compute raw_damage_rate = (prev_health - cur_hp) / (delta * max_hp), clamped to [0, 1]. Handle first-frame sentinel (_prev_health < 0 -> skip damage computation, set _prev_health = cur_hp). Compute raw_health_ratio = cur_hp / max_hp. Apply EMA: `self.ema_X = lerp(self.ema_X, raw_X, self.EMA_RATE)` where lerp(a,b,t) = a + (b-a)*t.

    For `_update_multipliers`: Per D-09, compute threat_level = 1.0 - ema_health_ratio. For DEFENSIVE types: target = lerp(MULT_MIN, MULT_MAX, threat_level), then lerp multiplier toward target at EMA_RATE, clamp to [MULT_MIN, MULT_MAX]. For OFFENSIVE types: target = lerp(MULT_MAX, MULT_MIN, threat_level), same lerp + clamp.

    Include a module-level `_lerp(a, b, t)` helper: `return a + (b - a) * t`.
